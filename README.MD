
# Chronomètre pour Gamebuino META

v1.1.0

## Présentation

Ce chronomètre n'est pas aussi fiable qu'un vrai ! Ne vous en servez donc pas pour chronométrer un 100 mètres ou un marathon !

Par défaut, on affiche le mode "CHRONO".
